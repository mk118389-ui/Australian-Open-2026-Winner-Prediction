# 🎾 Australian Open 2026 Winner Prediction

This project predicts the winner of the Australian Open 2026 using historical ATP match data (2020–2025).

## 🚀 Features
- Machine Learning model trained on ATP dataset
- Predicts win probabilities for players
- Interactive Streamlit web app
- Player-wise probability lookup
- Top contenders visualization

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## 📊 Dataset
ATP Australian Open matches (2020–2025)

## 🧪 How it works
1. Data preprocessing and feature engineering
2. Model training on past matches
3. Prediction of win probabilities
4. Deployment using Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
